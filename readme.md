# time.txt - simple, text-based time tracking app inspired by todo.txt project

## Build and install
Install via npm/yarn:
```
npm install time.txt -g
yarn add time.txt -g
```

Manual build:
```
git clone https://github.com/galczo5/time.txt.git
cd time.txt
npm install
npm install -g
```
